Galaxy Wars is a 2D space shooter game developed in C++ using SFML. The game demonstrates advanced OOP concepts through a modular and scalable architecture while delivering an engaging multi-level space combat experience.


- Gameplay Overview
The player controls a spaceship that must survive waves of enemies and asteroids, collect power-ups, and defeat a final boss. The difficulty increases across multiple levels, introducing new enemy behaviors and attack patterns.

-Features
Player-controlled spaceship with shooting mechanics
Multiple enemy types (Asteroid, ShootingEnemy, MissileEnemy, BossEnemy)
Power-ups (Shield, Double Score, Double Fire)
Multi-level progression system (Level 1 → Level 3 → Boss Level)
Collision detection system
Score tracking and file-based scoreboard
Menu system (Start, Pause, Game Over)
Boss battle with unique attack behavior

- OOP Concepts Implemented
Classes & Objects
Encapsulation
Inheritance (Single & Multilevel)
Polymorphism (Dynamic Binding)
Abstraction
Composition & Aggregation
Operator Overloading
File Handling (Scoreboard)
Exception Handling
Dynamic Memory Management
Virtual Destructors

- Technologies Used
C++17
SFML 2.5
OOP Principles

This project was developed to apply and demonstrate core and advanced OOP principles in a real-world game development scenario.
